# AutoRig MechRobot - Animation Rigging Scripts

## Overview

This repository contains a comprehensive collection of MEL (Maya Embedded Language) scripts for automated character rigging, specifically designed for robotic and mechanical character automation. These scripts were developed for the creation of complex mechanical rigs used in an indie animated series.

## Author

**Thai Ha Trieu (Hannah Trieu)**

All scripts in this repository were written by Thai Ha Trieu while working as a rigger on an indie film project with **AnimSchool**. She was responsible for rigging all of the robot characters in the series, creating efficient and reusable automation tools to streamline the rigging workflow.

### Project Link
Visit the animated series: [**The Mechwest Show**](https://mechwestshow.com/)

## Usage Notes

- These scripts are designed for **Autodesk Maya** and utilize MEL scripting
- Scripts should be loaded into Maya's script editor or added to a custom shelf
- Many scripts work together as part of a complete rigging pipeline
- Proper joint naming conventions are required for scripts to function correctly
- Some scripts have inter-dependencies (noted in header documentation)
